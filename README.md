# おみくじアプリ

このリポジトリは、Streamlitを使ったシンプルなおみくじアプリです。

## セットアップ

```bash
git clone https://github.com/<USERNAME>/<REPO>.git
cd <REPO>
python -m venv venv
. venv/bin/activate
pip install -r requirements.txt
```

## 実行方法

```bash
streamlit run app.py
```

ブラウザが自動で開き、おみくじアプリが表示されます。ボタンをクリックして結果をお楽しみください。
